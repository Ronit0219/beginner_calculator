# Beginner Calculator

A simple terminal-based calculator built to practice coding basics. It includes versions in both **C** and **Python**.

## ✨ Features
* Multiplies, divides, adds, and subtracts.
* Stops crashes if you type letters instead of numbers.
* Prevents errors if you try to divide by zero.

## 🚀 How to Run

### Python Version
Run this command in your terminal:
```bash
python calculator.py
```

### C Version
1. Compile the code:
   ```bash
   gcc calculator.c -o calculator
   ```
2. Run the program:
   ```bash
   ./calculator
   ```
